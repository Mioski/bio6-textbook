# Биология 6 класс — Интерактивный учебник

Интерактивный образовательный SPA-учебник по биологии для 6-го класса. Охватывает все 62 УТК темы программы.

## Запуск

```bash
# Разработка
npm install
npm run dev

# Сборка
npm run build
# Открыть dist/index.html в браузере (работает по file://)
```

## Архитектура

| Слой | Технология |
|---|---|
| UI | Vue 3 + Composition API (`<script setup>`) |
| Маршрутизация | Vue Router 4 + `createWebHashHistory` |
| Сборка | Vite 5 + `base: './'` |
| Данные | JSON-файлы в `static/bio6/` |
| Состояние | `provide/inject` + `localStorage` |

### Структура проекта

```
src/
  router/          — хэш-маршрутизатор
  services/bio6/   — ContentService, ProgressService, ChainService, NavigationService
  views/bio6/      — страницы (index, overview, node, base)
    components/    — переиспользуемые компоненты
      sims/        — интерактивные симуляторы
static/bio6/       — JSON-данные (9 файлов)
images/            — SVG-иконки
dist/              — готовая сборка
```

## Функциональность

### 4 «глубоких» темы (полные симуляторы)

| Тема | Симулятор |
|---|---|
| Растительная клетка | SVG-диссекция с 7 кликабельными слоями |
| Фотосинтез | 3 слайдера (свет / CO₂ / температура), анимация пузырьков O₂ |
| Строение семени | SVG-схема с reveal-механикой, 6 кликабельных частей |
| Жизненный цикл растения | Круговая SVG-анимация, 6 фаз |

### 58 «базовых» тем

Каждая включает:
- Крючок-вопрос (hook)
- Шпаргалка (cheatsheet): что это, факты, не путать, мнемоника
- Тренажёр-сортировка (bucket sort)
- Тренажёр-соответствие (match)
- Глоссарий с латынью и примерами
- Разбор типичных ошибок (misconceptions)
- Финальный квиз (3–6 вопросов с объяснениями)
- «Знаете ли вы» (занимательный факт)
- Мостик к следующей теме (chain)

## Дизайн

Мобильная тёмная тема (360–414 px, portrait-first).

Цветовая система:
- `--bg: #0d1117` — основной фон
- `--accent-gold: #e8c891` — акцент
- `--accent-blue: #a8c5d8` — вторичный акцент
- `--correct: #8cc882` — правильный ответ
- `--wrong: #e8a070` — неправильный ответ
